\b;\button 153; Robot orgaShooter
\image botor 8 8;
Robot à roues avec un canon orgaShooter.

Le canon orgaShooter est plus efficace que le \l;canon shooter\u object\botfr;. Il projette de petites boules de matière organique corrosive. Une \l;pile normale\u object\power; permet de tirer onze fois.

La propulsion avec des roues est le mode standard, rapide, consommant peu d'énergie et adapté aux terrains relativement plats. Lorsque le relief devient trop accidenté, il faut opter si possible pour un \l;robot volant\u object\botoj; ou, si le vol est impossible, pour une propulsion à \l;chenilles\u object\botoc;.

\t;\l;Catégorie\u cbot\category; de l'objet
\c;WheeledOrgaShooter\n;

\t;Programmation
Pour déplacer le robot par programmation, il faut utiliser les instructions \c;\l;move\u cbot\move;\n;, \c;\l;turn\u cbot\turn;\n;, \c;\l;motor\u cbot\motor;\n;, \c;\l;goto\u cbot\goto;\n;.

Pour actionner le canon par programmation, il faut utiliser l'instruction \c;\l;fire\u cbot\fire;\n;. L'instruction \c;\l;aim\u cbot\aim;\n; permet de l'orienter vers le haut ou vers le bas.
