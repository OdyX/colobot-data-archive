\b;Exercice
Ce premier exercice est vraiment très simple. Il faut déplacer le robot d'entraînement jusqu'à la croix bleue en avançant tout droit d'une longueur de 20 mètres.

\b;Marche à suivre
1) Programmer le robot. Pour cela, il faut cliquer sur le bouton \button 22;.
2) Taper le code du programme, puis cliquer le bouton <D'accord>.
3) Exécuter le programme \button 21; et observer le résultat.

Le corps du programme est constitué de¦:
\c;
\s;extern void object::Go( )
\s;{
\s;	\n;à compléter ...\c;
\s;}
\n;
Vous ne devez pas modifier ces caractères, pour l'instant. Les instructions doivent être données entre les accolades \c;{ }\n;.

\b;Programme
L'instruction \c;move(20);\n; avance de 20 mètres. Le programme qui doit être écrit est donc¦:
\c;
\s;extern void object::Go( )
\s;{
\s;	move(20);
\s;}
\n;
\b;Remarques
Attention de tout taper avec des minuscules¦!
Une instruction est terminée par un point-virgule.

Si vous êtes bloqué, vous pouvez toujours regarder la solution: sélectionnez le programme \c;Solution\n; tout en bas à gauche, puis cliquez sur le bouton avec les accolades \button 22;. Vous pouvez aussi exécuter le programme représentant la solution avec le bouton \button 21;.

En cas de tentative infructueuse, vous pouvez remettre le robot à la position initiale avec le bouton \button 59;.
